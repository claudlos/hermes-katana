## Part B: Vault Memory Safety (src/hermes_katana/vault/store.py)

### 7. Secure key zeroing on __del__ (CRITICAL — GAP 2.1)
Add __del__ method that zeros _master_key bytes. Use ctypes to overwrite:
```python
import ctypes
def _zero_key(self):
    if hasattr(self, '_master_key') and self._master_key:
        buf = (ctypes.c_char * len(self._master_key)).from_buffer_copy(self._master_key)
        ctypes.memset(buf, 0, len(self._master_key))
        self._master_key = None
```
