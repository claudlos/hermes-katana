- Class: ToolRegistry. Handlers live in `self._tools[name]` (ToolEntry).

## Gotchas for patch writers
- dispatch is SYNC not async → if katana middleware is async, wrap with
  `_run_async` (import from model_tools) before calling.
- dispatch returns JSON str — middleware MUST return a JSON str too if
  it short-circuits (e.g. block), not a dict.
- terminal_tool has many env backends; picking `_create_environment` as
  anchor covers all of them in one shot.
- banner output is Rich-console-based; append to panel, don't print.
- BasePlatformAdapter.handle_message is async; scan hook must be awaitable
  or sync non-blocking.
- There is no `hermes/__init__.py` package anymore — installer must write
  to repo-root paths, not `hermes/<subpath>`.
